## mPlayer :headphones:

### Features

- Register
- Forgot Password
- Login
- Handle User Profile
- Follow other accounts (users)
- Play Music (Streaming)
- Playlist managements
- Copy other user's playlists
- Listen to other user's playlists
- Support for FLAC, MP3, etc
- Read music lyrics
- Album covers
- etc...

### Tech Stack

- Java 17
- Spring Boot
- Docker
- PostgreSQL
- RabbitMQ
